# Agent IA

Un agent est un ensemble de fonctionnalités construites **au-dessus d'un LLM**. Sans le LLM, l'agent n'est rien — c'est le "cerveau".

## Composants principaux

1. **[[Planification (agents IA)|Planification]]** — décomposer des tâches complexes, gérer des sous-objectifs, s'auto-critiquer
2. **[[Mémoire (agents IA)|Mémoire]]** — court terme (contexte) et long terme (bases vectorielles)
3. **[[Outils (agents IA)|Outils]]** — interagir avec le monde extérieur (APIs, bases de données, services)

## Frameworks courants

- LangChain
- LlamaIndex
- CrewAI
