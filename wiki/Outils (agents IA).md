# Outils (agents IA)

Troisième composant d'un [[Agent IA]] — permettent d'**interagir avec le monde extérieur**.

## Usages

- **Récupérer de l'information** absente des poids du modèle (ex : résultats sportifs en temps réel, cours de bourse)
- **Agir sur des systèmes externes** (ex : insérer des données en base, envoyer un email, appeler une API)

## Exemples concrets

- Appel d'API REST
- Requêtes sur une base de données (MySQL, PostgreSQL...)
- Recherche web
- Exécution de code

Les outils sont ce qui transforme un LLM passif en agent capable d'**agir**.
