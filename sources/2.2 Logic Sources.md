Data tells you what IS. Logic tells you how to THINK about it.

![](https://everpath-course-content.s3-accelerate.amazonaws.com/instructor%2F6rv9eqj974u5ivyxlj7ndbkii%2Fpublic%2F1777640290%2FLogic+Sources+Cropped.1777640290263.png)

Logic sources in Foundry span a wide range of complexity. At the simplest level, they might be rules captured in a spreadsheet, SOPs or tribal knowledge of your organization, all the way to sophisticated ML models for forecasting, third-party route optimizers, custom algorithms.  
  
**Integration tools** support this full range. Models can be developed inside the platform or brought in from outside — containerized runtimes allow teams to package their existing model code in Docker containers and deploy them directly into the Foundry environment.  
  
**MMDP for compute** extends the "any compute" philosophy to logic and transformation workloads. Foundry's compute-agnostic build framework supports mixing-and-matching of third-party compute runtimes.
