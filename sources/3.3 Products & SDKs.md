Finally, platform's capabilities can be packaged into products that reach end users — whether those are internal operators, external customers, or other systems.![](https://everpath-course-content.s3-accelerate.amazonaws.com/instructor%2F6rv9eqj974u5ivyxlj7ndbkii%2Fpublic%2F1777644869%2FP%26SDKs+cropped.1777644868735.png)

**The Ontology SDK (OSDK)** is a generated software development kit that gives developers programmatic access to the Ontology from their own development environment. Developers can create custom React applications, Python backends, Java services, or integrations in any language via OpenAPI — all powered by the Ontology's data, logic, and actions.  
  
In effect, the OSDK turns your Ontology into an SDK of your business. Agentic workflows built with the OSDK can be integrated into existing applications — your organization's existing tools don't need to be replaced; they can be enhanced with Ontology-powered capabilities.  
  
Beyond APIs and SDKs, the platform supports integration through streaming and batch connectors with export plugins, webhooks for real-time write-back to external systems, and external transforms that allow data transformations to run outside of Foundry while integrating results back in.
